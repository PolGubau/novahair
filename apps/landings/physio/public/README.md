# Public Assets

Esta carpeta contiene los assets públicos de la landing de PhysioCare.

## Imágenes requeridas

Para que la landing funcione correctamente, necesitas añadir las siguientes imágenes en la carpeta `public/images/`:

- `1.webp` - Imagen principal del hero (izquierda)
- `2.webp` - Imagen central del hero
- `3.webp` - Imagen principal del hero (derecha)
- `4.webp` - Imagen para parallax
- `5.webp` - Imagen para parallax
- `6.webp` - Imagen para parallax
- `7.webp` - Imagen para parallax
- `8.webp` - Imagen para parallax

## Recomendaciones

- Formato: WebP para mejor rendimiento
- Tamaño recomendado: 800x1200px para imágenes verticales
- Optimiza las imágenes antes de subirlas
- Usa imágenes relacionadas con fisioterapia: tratamientos, instalaciones, pacientes felices, etc.

## Favicon

Añade también un `favicon.svg` en la raíz de `public/`
